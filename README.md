# IoT-Arduino-UNO-R4-WIFI-Weather-Station-with-DHT11-Sensor
This project Uses Arduino Cloud to monitor the Temperature and Humidity readings from a DHT11 sensor Through an Arduino UNO R4 WIFI Wirelessly using IoT.
